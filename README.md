## 🚀 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/drakeshnag-rjo/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in a browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. Access at `http://localhost:8000`
